poeNavigator


This app shows last entered location layouts and tips. Use it for navigation and faster leveling.

1. Enable "Aero" Windows theme.
2. Run English PoE in "Windowed Fullscreen" mode.
3. Set path to PoE folder in app options.

Optional:
4. Set hotkey in options. If hotkeys don't work: run app as admin or don't run PoE as admin.
5. Check "game part" in notify icons menu ("Part 1": Acts 1-5. "Part 2": Acts 6-10).
